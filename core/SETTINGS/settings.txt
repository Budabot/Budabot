<header> :::::: Settings command :::::: <end>

<highlight>To see a list of all settings on the bot<end>
<symbol>settings
Note: When a setting is editable, it will include a 'Modify' link next to the setting name.

<highlight>To see info about a particular setting<end>
<symbol>settings change 'setting name'

<highlight>To save a value for a particular setting<end>
<symbol>settings save 'setting name' 'value'
