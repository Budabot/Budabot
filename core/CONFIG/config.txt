To show a list of all the modules that can be configured:
<highlight><tab><symbol>config<end>

To configure a specific module:
<highlight><tab><symbol>config <i>MODULE_NAME</i><end>

To configure a specific command:
<highlight><tab><symbol>config cmd <i>command</i><end>

To enable or disable all commands for a certain channel:
<highlight><tab>config cmd enable|disable all|guild|priv|msg
