To restart the bot:
<highlight><tab><symbol>restart<end>

To shutdown the bot:
<highlight><tab><symbol>shutdown<end>

To see an overview of the bot systems:
<highlight><tab><symbol>system<end>

To execute a query on the database:
<highlight><tab><symbol>executesql <i>query</i><end>

Note: 'query' should be an INSERT, UPDATE, or DELETE only as it won't return a result set

To query the database:
<highlight><tab><symbol>querysql <i>query</i><end>

Note: 'query' should be a SELECT

To show the last lines in a log file:
<highlight><tab><symbol>logs<end>
