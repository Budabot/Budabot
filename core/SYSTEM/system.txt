<header> :::::: System Commands :::::: <end>

<highlight>To restart the bot:<end>
<tab><symbol>restart

<highlight>To shutdown the bot:<end>
<tab><symbol>shutdown

<highlight>To manually load changes from the config file:<end>
<tab><symbol>reloadconfig

<highlight>To see an overview of the bot systems:<end>
<tab><symbol>system

<highlight>To execute a query on the database:<end>
<tab><symbol>executesql 'query'
Note: 'query' should be an INSERT, UPDATE, or DELETE only as it won't return a result set

<highlight>To show the last lines in a log file:<end>
<tab><symbol>logs
