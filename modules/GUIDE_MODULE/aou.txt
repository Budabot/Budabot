To search for a guide on AO-Universe:
<highlight><tab><symbol>aou 'search'<end>
Note: this will search the name, category, and description for matches.

To search for a guide on AO-Universe and include all results:
<highlight><tab><symbol>aou all 'search'<end>
Note: this will search the name, category, and description as well as the guide body for matches.

To view a guide on AO-Universe:
<highlight><tab><symbol>aou 'guide id'<end>
