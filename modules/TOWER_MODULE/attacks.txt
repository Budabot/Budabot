<highlight>To see the last 20 attack messages:<end>
<tab><symbol>attacks

<highlight>To see page 2 of the last attack messages:<end>
<tab><symbol>attacks 'page number'

<highlight>To see the last 20 attack messages for a certain site:<end>
<tab><symbol>attacks 'playfield' 'site number'

<highlight>To see page 2 of the last attack messages for a certain site:<end>
<tab><symbol>attacks 'playfield' 'site number' 'page number'

<highlight>To see the last 20 attack messages for specific person:<end>
<tab><symbol>attacks player 'character name'

<highlight>To see page 2 of the last attack messages for specific person:<end>
<tab><symbol>attacks player 'character name' 'page number'

<highlight>To see the last 20 attack messages for specific guild:<end>
<tab><symbol>attacks org 'exact guild name'

<highlight>To see page 2 of the last attack messages for specific guild:<end>
<tab><symbol>attacks org 'exact guild name' 'page number'

Note: You can use '%' as a wildcard in guild and player names
