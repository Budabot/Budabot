<highlight>To see the last 20 victory messages:<end>
<tab><symbol>victory

<highlight>To see page 2 of the last victory messages:<end>
<tab><symbol>victory 'page number'

<highlight>To see the last 20 victory messages for a certain site:<end>
<tab><symbol>victory 'playfield' 'site number'

<highlight>To see page 2 of the last victory messages for a certain site:<end>
<tab><symbol>victory 'playfield' 'site number' 'page number'

<highlight>To see the last 20 victory messages for specific person:<end>
<tab><symbol>victory player 'player name'

<highlight>To see page 2 of the last victory messages for specific person:<end>
<tab><symbol>victory player 'player name' 'page number'

<highlight>To see the last 20 victory messages for specific guild:<end>
<tab><symbol>victory org 'exact guild name'

<highlight>To see page 2 of the last victory messages for specific guild:<end>
<tab><symbol>victory org 'exact guild name' 'page number'

Note: You can use '%' as a wildcard in guild and player names
