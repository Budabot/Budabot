To show the current heal assist macro:
<highlight><tab><symbol>heal<end>

To set a new heal assist macro:
<highlight><tab><symbol>heal 'player'<end>

To set a new heal assist macro with multiple players:
<highlight><tab><symbol>heal 'player1' 'player2' 'player3' ...<end>

To clear the heal assist macro:
<highlight><tab><symbol>heal clear<end>
