<header> :::::: Help on Timer Commands :::::: <end>

<orange>Note: You can use <symbol>timers and <symbol>timer interchangeably.<end>

<highlight>To show current timers running:<end>
<tab><symbol>timers

<highlight>To set a primary timer:<end>
<tab><symbol>timer 'time'

<highlight>To set a named timer:<end>
<tab><symbol>timer 'time' 'Timername'

<highlight>To remove a timer(only your own):<end>
<tab><symbol>timer rem 'Timername'

<highlight>To create a repeating timer:<end>
<tab><symbol>rtimer 'intial time' 'repeating time' 'name'

See <a href='chatcmd:///tell <myname> help budatime'>budatime</a> for info on the format of the 'time' parameter.
