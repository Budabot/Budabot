<header>::::: Alternative Char Commands (admin) :::::<end>

<highlight>Adding alts to a character<end>
<tab><symbol>altsadmin add 'alt name' 'main toon'

<highlight>Remove alts from a character<end>
<tab><symbol>altsadmin rem 'alt name' 'main toon'
<tab>use <symbol>alts 'alt name'
<tab>if you are unsure of who the main is.

<highlight>Export all alts to a file<end>
<tab><symbol>altsadmin export 'filename'
<tab>the file will be placed in the bots main directory

<highlight>Import alts from a file<end>
<tab><symbol>altsadmin import 'filename'
<tab>the file will be loaded from the bots main directory
