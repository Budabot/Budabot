<header>::::: Help on Tower Commands. :::::<end>

<red>Remember that the bot needs at least Squad Commander to be able to see Tower Messages<end>

<highlight>To see the last attack messages:<end>
<tab><symbol>battle

<highlight>To see the last attack messages for specific person/org/zone:<end>
<tab><symbol>battle lucier

<highlight>To see the last attack messages with a wildcard search:<end>
<tab><symbol>battle lucier%

<highlight>To see the last tower battle result messages:<end>
<tab><symbol>victory

<highlight>To see the last tower battle result messages from a specific org:<end>
<tab><symbol>victory my org

<highlight>To see the last tower battle result with a wildcard search:<end>
<tab><symbol>victory my org%
