<header>::::: DB1/DB2 Instance Loot Module :::::<end>

This module is designed to assist in adding DB1/DB2 loot to a loot list.

<highlight>To View Loot<end>
<tab><symbol>db1 - DB1/Old DB/Twister
<tab><symbol>db2 - DB2/New DB/Inside the Machine

<tab>Click on item icons to view item references
<tab>db1 - Click the 'Add to Loot List' Link to add an item
<tab>db2 - Click the 'Create Loot List' on the Armor item which dropped and full DB2 loot table will be loaded along with that armor item
