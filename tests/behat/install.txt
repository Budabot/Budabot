To install Behat:
 1. Open command prompt/bash/etc...
 2. cd to this directory
 3. Get Behat:
     - Unix/Linux:
	    1. curl http://getcomposer.org/installer | php
        2. php composer.phar install --prefer-source
     - Windows:
	    1. Download and install http://getcomposer.org/Composer-Setup.exe
		2. composer install --prefer-source
